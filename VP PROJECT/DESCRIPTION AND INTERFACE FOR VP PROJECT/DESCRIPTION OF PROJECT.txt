PROJECT NAME : Steganography



DESCRIPTION

The basic purpose of this project is to implement the procedure of image steganography.

The code will be able to take an image from the user's computer by browsing for the image in the computer. The image will already 
be present beforehand for hiding the information on it.

There will be two options for encrypting the message in the image and for decrypting the message in the image.
The message will also be generated by the user that is needed to be encrypted into the image.
